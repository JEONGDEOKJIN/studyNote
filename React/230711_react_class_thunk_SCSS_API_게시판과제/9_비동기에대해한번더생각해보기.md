여기서 중요한 부분은 자바스크립트의 이벤트 루프(Event Loop)입니다. 이벤트 루프는 대기 중인 작업이 없을 때, 콜백 큐에서 함수를 가져와 실행합니다.

그래서 await 키워드를 만나면, 해당 비동기 작업이 완료될 때까지 기다리는 동시에, 그 완료 후에 실행할 작업(일종의 콜백 함수)를 콜백 큐에 등록하게 됩니다. 그런 다음에 다음 코드를 계속 실행하게 됩니다.

이런 방식으로, await 키워드를 사용한 코드는 동기적으로 보이지만 실제로는 비동기적으로 동작하며, 사용자에게는 여러 작업이 동시에 실행되는 것처럼 보이게 됩니다.
