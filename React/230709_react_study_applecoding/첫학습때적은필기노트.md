### ◼ state 문법을 왜 쓰나? 언제 쓰면 좋나? 

```
- '⭐꽂을 데이터⭐' 가 변경되면 'html' 이 '자동 ⭐재렌더링⭐' 되면서, '⭐html에 자동 반영⭐' 해줌 (⭐데이터 바인딩⭐)
```
